차트 라이브러리 비교 — recharts vs lightweight-charts

시뮬레이션 데이터: 250 영업일치 일봉 페어 잔차 (AR(1), half-life ≈ 3.5일) + 최근 30일 강한 음의 drift. 실제 stat-arb-engine `/pairs/detail` 응답 구조 그대로 (spread_series, z, histogram).

n=250 β=+2.58 R²=0.93 ADF=-4.99 half-life=3.4일 현재 z=-3.52

1. recharts (React, SVG 기반 — LENS dividends 페이지 사용중)

스프레드 시계열 (잔차)

z-score 시계열 + ±1·±2 밴드

잔차 히스토그램 + 현재 위치

[참고]

recharts는 React 컴포넌트 기반 SVG 차트. ResponsiveContainer로 컨테이너 폭에 자동 맞춤. hover 툴팁, ReferenceLine으로 밴드 점선, ReferenceDot로 현재점 마킹 표준 제공. LENS의 다른 페이지(dividends)와 동일 스타일로 통일 가능.

2. lightweight-charts (TradingView, Canvas — 대량 점 빠름)

스프레드 시계열 (잔차)

z-score 시계열 + ±1·±2 밴드

잔차 히스토그램 (Histogram series)

[참고]

lightweight-charts는 Canvas 기반. 휠 스크롤 줌/팬이 매우 부드러움 — 수만 점도 60fps. ±밴드는 별도 LineSeries로 점선. 현재점 마킹은 PriceLine으로 가능. 트레이딩뷰 스타일 — LENS 다른 페이지와 시각 어울리지 않을 수도 있음.

실제 LENS 환경에서는 stat-arb-engine `/pairs/detail` 응답을 그대로 사용. recharts는 LENS 다른 차트와 일관, lightweight-charts는 트레이딩뷰 스타일 + 대량 점에서 빠름.
